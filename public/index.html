SPASSU Tecnologia . JIRA API Automation

This is an Express application build to provide endpoints for handling JIRA API requests


Available endpoints

  • getprojects

    Returns all project names and keys from Atlassian API

  • getsprintreport/{projectKey}

    Returns Sprint report data for last closed Sprint in a given project key.

    Includes completed tasks (key,url,summary,description), startDate, endDate, completedDate, sprintName and a few more fields.
